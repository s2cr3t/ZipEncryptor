# ZIP文件加密工具 - 项目说明文档

## 项目概述

这是一个企业级的ZIP文件加密工具，使用C++和Qt框架开发，提供现代化的用户界面和高效的加密功能。该工具允许用户指定目录，批量加密ZIP文件，并提供实时的加密进度显示。

## 功能特点

- 现代化UI设计，包含毛玻璃效果，符合Windows平台主流审美
- 中文界面，操作简单直观
- 支持批量选择和加密多个ZIP文件
- 实时显示加密进度
- 密码输入验证，确保加密安全
- 自动保存上次使用的目录路径
- 支持取消正在进行的加密操作

## 技术实现

- 开发语言：C++
- UI框架：Qt 5
- 加密方法：使用AES256算法对ZIP文件进行加密
- 多线程处理：确保UI响应性和加密效率
- 异常处理：完善的错误处理机制

## 编译说明

### 环境要求

- Windows 10/11
- Qt 5.15或更高版本
- C++编译器（如MSVC 2019或更高版本）
- 7-Zip命令行工具（用于ZIP加密）

### 编译步骤

1. 安装Qt开发环境和Qt Creator
2. 打开Qt Creator，选择"打开项目"
3. 导航到项目目录，选择项目文件（.pro文件）
4. 配置项目构建设置
5. 点击"构建"按钮编译项目
6. 编译完成后，点击"运行"按钮测试应用

### 部署步骤

1. 使用Qt的windeployqt工具收集所需的DLL和资源文件
2. 确保目标系统安装了7-Zip或将7z.exe和必要的DLL打包到应用程序目录
3. 创建安装程序（可选）

## 使用说明

1. 启动应用程序
2. 点击"浏览"按钮选择包含ZIP文件的目录
3. 从列表中选择需要加密的ZIP文件（可多选）
4. 输入加密密码并确认
5. 点击"开始加密"按钮开始加密过程
6. 加密完成后，加密后的文件将保存在原目录中，文件名带有"_encrypted"标记

## 注意事项

- 请妥善保管加密密码，密码丢失将无法恢复加密文件
- 加密过程中请勿关闭程序，如需中断请使用"取消"按钮
- 加密后的文件与原文件存放在同一目录，请确保磁盘空间充足
